## Architecture

- `src/core/` defines shared config, database helpers, and Pydantic domain contracts.
- `src/chain/` contains Solana/Jupiter/wallet integration placeholders plus a pre-live Jito block-engine scaffold for MEV-aware bundle request building and injectable submission testing.
- `src/risk/` contains token risk checks, aggregate scoring, standalone buyer funding-source analysis for sybil/bundle detection, a standalone honeypot sell-simulation adapter for deeper read-only backend checks, signal-aware token enrichment for bounded paper-cycle runs, a RugCheck-assisted scorer path for authority/holder/honeypot enrichment, and a parallelized scorer precheck stage that can run RugCheck plus funding analysis together before optional holder lookup and optional honeypot-simulation fallback.
- `src/execution/` defines the execution adapter contract plus paper and live adapter implementations, with a disabled-by-default live Jupiter submission helper that can optionally route serialized swaps through Jito and fall back to RPC via injected submitters while leaving real live trading disabled.
- `src/signals/` defines async signal-source contracts; `aggregator.py` safely starts/stops/polls sources, clusters same-mint signals by time window, boosts multi-source composites, and best-effort logs ranked opportunities, while `whale_tracker.py`, `pump_fun.py`, `onchain.py`, and `twitter.py` provide Helius, PumpPortal, DexScreener, and Twitter/X-backed signal feeds.
- `src/strategy/` contains decision, portfolio, position, fixed-exit helpers, gated dynamic-exit runtime hooks for calibrated volume-decay exits, optional market-regime classification helpers for later risk tuning, a default-off decision metadata hook that can surface regime summaries without changing entry or sizing behavior, and a position-manager option for session-local paper evaluation that ignores previously persisted positions without changing live or default paper behavior.
- `src/monitoring/` contains lightweight health, alert, and dashboard helpers for runtime visibility.
- `src/cli.py` exposes operator commands including health/config inspection and a bounded `paper-cycle` runtime that evaluates ranked aggregated opportunities from pump.fun, whale, on-chain, and Twitter sources, persists paper trades/positions, and prints a safe summary before terminating, with strict and discovery paper-only risk profiles plus a manual-only Grok social-check prompt export for the top discovery candidates using safe normalized fields only.
